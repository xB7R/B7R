## About Me

I am a cybersecurity student interested in building, testing, and documenting secure IT environments. My work focuses on network security, GRC, SOC operations, incident response concepts, and practical lab-based validation.

I use GitHub to document my academic and practical cybersecurity projects, including secure enterprise networks, firewall configurations, IDS/IPS validation, VPN design, AAA/RBAC access control, and security monitoring workflows.

## Featured Project

### Secure Enterprise Network Infrastructure Project

A Defense-in-Depth Networking Security project focused on designing and validating a secure enterprise network infrastructure.

<table>
  <tr>
    <th>Project Area</th>
    <th>Implemented Controls</th>
    <th>Validation</th>
  </tr>
  <tr>
    <td><b>Segmentation</b></td>
    <td>VLANs for Marketing, Sales, HR, and Management.</td>
    <td>Verified using VLAN database and SVI configuration outputs.</td>
  </tr>
  <tr>
    <td><b>L2/L3 Hardening</b></td>
    <td>DHCP Snooping, DAI, Port Security, BPDU Guard, Root Guard, ACLs, and OSPF authentication.</td>
    <td>Validated using command outputs and traffic behavior testing.</td>
  </tr>
  <tr>
    <td><b>VPN</b></td>
    <td>Dual-hub DMVPN with IPsec encryption.</td>
    <td>Verified using NHRP registration and tunnel status outputs.</td>
  </tr>
  <tr>
    <td><b>IDS/IPS</b></td>
    <td>Cisco IOS IPS with alert and deny actions.</td>
    <td>Validated using scan activity, IPS alerts, and packet drop evidence.</td>
  </tr>
  <tr>
    <td><b>Firewall HA</b></td>
    <td>ASA Active/Active failover with NAT, ACLs, and security zones.</td>
    <td>Tested through failover validation and interface monitoring.</td>
  </tr>
  <tr>
    <td><b>AAA/RBAC</b></td>
    <td>AdminView, AuditView, and UserView role-based access.</td>
    <td>Validated by testing different user permissions.</td>
  </tr>
</table>
