## Featured Work

### Secure Enterprise Network Infrastructure

A university cybersecurity project focused on designing and validating a secure enterprise network using a Defense-in-Depth approach.

**Key areas covered:**

- VLAN-based network segmentation
- Layer 2 and Layer 3 hardening
- DMVPN with IPsec for secure branch connectivity
- Cisco IOS IPS detection and prevention
- ASA firewall high availability
- NAT and ACL enforcement
- AAA/RBAC access control
- Practical validation using screenshots and command outputs
